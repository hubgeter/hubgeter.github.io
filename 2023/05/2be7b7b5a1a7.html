note

list 是双向循环链表 list里有一根指针 指向的是list_node（ 三根指针 pre nxt data）

array ：数组

deque ： 二维数组指针 ，两个iterator 分别管理两个数组 （first end cur 在数组中的位置），数组大小 ； deque 控制中心是一个vector , 进行扩充时候会把数据copy到中端.

算法操作容器，通过迭代器 ，算法根据迭代器的性质，来实现具体的操作 算没提问迭代器，迭代器回答

迭代器（泛化的指针）提供五种associated types
iterator_category: 移动性质 可以随便移动/只能前进/后退
value_type : 所指的元素的type . int string ….
difference_type : 两个iterator 之间的距离应该用什么类型来表现 int long…
reference
pointer
对于普通指针，通过萃取机得到五个associate type

仿函数 只为算法而服务 提供 + - ^ … 这些东西

适配器
container stack queue priority_queue
