按引用捕获
一旦由lambda 式所创建的闭包越过了该局部变量或形参的生命期，闭包内的引用都会空悬，捕获只能针对在创建lambda式的作用域内可见的非静态局部变量。

按值捕获
按值默认捕获对于类而言：捕获其this指针 ，lambda 闭包的存活与它含有this指针副本的对象的生命周期绑定，可将其想捕获的成员变量复制到局部变量中之后，再进行按值捕获对应的局部变量。
